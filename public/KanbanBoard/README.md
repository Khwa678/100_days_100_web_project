# 📋 Kanban Board Web Application

## Overview

The Kanban Board Web Application is a simple and interactive task management tool that helps users organize their work visually. Tasks can be created, edited, deleted, and moved across different workflow stages using drag-and-drop functionality.

This project is built entirely using HTML, CSS, and JavaScript and stores data locally in the browser using Local Storage, eliminating the need for a backend.

---

## Features

### Core Features

* ✅ Create new tasks
* ✅ Edit existing tasks
* ✅ Delete tasks
* ✅ Drag and drop task cards
* ✅ Multiple workflow stages:

  * To Do
  * In Progress
  * Review
  * Completed
* ✅ Responsive design
* ✅ Local Storage support

### Additional Features

* 🌙 Dark/Light Mode
* 🏷️ Task Priority Labels (Low, Medium, High)
* 💾 Persistent task storage
* 📱 Mobile-friendly interface

## How to Run

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project folder:

```bash
cd KanbanBoard
```

3. Open `index.html` in your browser.

No additional installation or dependencies are required.

---

## Usage

1. Click the **Add Task** button.
2. Enter the task title and select a priority.
3. Save the task.
4. Drag and drop tasks between workflow columns.
5. Edit or delete tasks whenever needed.
6. Switch between light and dark themes using the theme toggle button.
