# Expense Tracker (Budget Tracker)

A modern and responsive Expense Tracker application that helps users manage their income, expenses, and budgets efficiently.

## Features

* Add, edit, and delete transactions
* Track income and expenses
* Budget management
* Monthly financial insights
* Dark and Light mode support
* Responsive design for desktop and mobile devices
* Local storage support for persistent data

## Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript

## Project Structure

```text
Budget Tracker/
├── index.html
├── style.css
├── script.js
└── assets/
```

## Getting Started

### Prerequisites

Any modern web browser such as:

* Google Chrome
* Microsoft Edge
* Mozilla Firefox

### Run Locally

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project directory:

```bash
cd "Budget Tracker"
```

3. Open `index.html` in your browser.
